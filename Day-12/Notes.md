# Day 12 Notes

## Problems Covered

1.  Aggressive Cows

-   Pattern: Binary Search on Answer
-   Goal: Maximize minimum distance

2.  Kth Element of Two Sorted Arrays

-   Pattern: Binary Search on Partitions
-   Goal: Find kth element without merging

3.  Allocate Minimum Pages

-   Pattern: Binary Search on Answer
-   Goal: Minimize maximum pages allocated

## Key Observation

-   Maximize minimum → Binary Search on Answer
-   Minimize maximum → Binary Search on Answer
-   Two sorted arrays → Binary Search on Partition
